Display

Display es una propiedad que modifica el comportamiento de las cajas, ya no modifica la interaccion de la caja sino el comportamiento en si de una caja en particular hablamos de relaciones


Dentro de las propiedades de display tenemos:
se usa Display BLOCK: ocupa todo el bloque y no se puede poner mas bloques
se usa Display INLINE: ocupa solo lo que se escriba y permite poner un bloque al lado del otro
se usa Display INLINE-BLOCK: podemos modificar la altura y anchura y permite poner un bloque al lado del otro

No se usa Display TABLE: que se comporte como tabla
No se usa Display INLINE-TABLE: que se comporte como tabla en linea
No se usa Display LIST-ITEM:
No se usa Display TABLE-CELL: que se comporte como la celda de una tabla
No se usa Display TABLE-ROW: que se comporte como cada fila de una tabla
No se usa Display TABLE-COLUMN: que se comporte como cada columna de una tabla

se usaDisplay FLEX:
se usaDisplay GRID:
