Importar css

@import:Permite cargar ficheros .css externos, lee las lineas de codigo e incorporarlo al archivo actual. Estas reglas CSS suelen indicar en las primeras lineas de fichero, ya que deben figurar antes de las otras reglas CSS o contenido CSS similar.

Tipos de importacion | Descripci√≥n

@import url(); | importamos una hoja de estilos css externa.

@import url(); mediaquery | importamos una hoja de estilo css solo si coincide con el mediaquery indicado.

@import url(); supports (condicion) | importamos una hoja de estilo css solo si el navegador soporta la condicion.

@import url(); layer | importamos una hoja de estilo css y la colocamos en la capa nombre.

@import url(); layer() | importamos una hoja de estilo css y la colocamos en una nueva capa anonima.

Formato del import

@import url("estilos.css"); Normal

@import "estilos.css"; Sin url()

@import url("layout/header.css"); dentro de una carpeta
