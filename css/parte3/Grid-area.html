Primero vamos a definir el header, el main, el aside y el footer.
Para trabajar con areas en vez de utilizar grid-template-columns y grid-template-rows utilizamos grid-template-areas.
Con grid-template-areas modificamos el anchor de toda nuestra pagina de la siguiente manera. grid-template-areas: "header header header" "aside main main" "footer footer footer" ;
