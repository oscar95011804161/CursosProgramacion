Curso de tailwind

Tailwind CSS es un framework de CSS basado en un enfoque de utilidades para la creación de interfaces de usuario. A diferencia de frameworks como Bootstrap, que ofrecen componentes predefinidos y estilos más opinativos, Tailwind se centra en proporcionar una amplia gama de clases de utilidad que se pueden combinar para construir cualquier diseño que imagines.
